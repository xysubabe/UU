# TabBar 图标资源

## 需要的图标文件

请将以下图标文件放置在 `assets/images/tab/` 目录下：

- `home.png` (首页图标 - 未选中状态)
- `home-active.png` (首页图标 - 选中状态)
- `order.png` (订单图标 - 未选中状态)
- `order-active.png` (订单图标 - 选中状态)
- `user.png` (我的图标 - 未选中状态)
- `user-active.png` (我的图标 - 选中状态)

## 图标规格

- 建议尺寸: 81px × 81px
- 格式: PNG
- 模式: 建议 image 组件 mode="aspectFit"
- 颜色:
  - 未选中: 灰色 (#999999)
  - 选中: 蓝色 (#4A90E2)

## 图标内容建议

- 首页: 房子图标 🏠
- 订单: 文档图标 📋 或 订单图标 📦
- 我的: 用户图标 👤 或 个人图标 🧑
